# WASD StreamControl
A completely reworked version of StreamControl using electron.

## Hotkeys
* Player 1 score up > F20
* Player 1 score down > CF21
* Player 2 score up > F22
* Player 2 score down > F23
* Swap players > F19
* Save changes  > F24
